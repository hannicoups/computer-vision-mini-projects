# OpenCV Image Processing using Python

## Description
This project contains several basic computer vision and image processing implementations using Python and OpenCV.

## Features
- Image access and processing
- Color detection
- Shape detection
- Camera input processing
- Object visualization using OpenCV

## Technologies
- Python
- OpenCV
- NumPy

## Files
- `opencv.py` → Basic OpenCV operations
- `color_detection.py` → Color detection implementation
- `shapeDetection.py` → Shape detection implementation
- `aksesFoto.py` → Image access and processing

## Topics Covered
- Computer Vision
- Image Processing
- Object Detection
- Color Segmentation
- Shape Recognition
